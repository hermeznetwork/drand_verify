# Verification of the drand round

## Usage

```
git clone https://github.com/hermeznetwork/drand_verify.git
cd drand_verify
npm install
```

To get and verify the the random of a specific round

```
node main.js 100000
```

To get the last generated random

```
node main.js latest
```
